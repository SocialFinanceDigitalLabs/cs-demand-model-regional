Using the Regional Demand Modelling Tool


This tool uses SSDA903 returns from REGION that was collected as part of the REGION DATA SHARING. The data stored is from {{ start_date }} to {{ end_date }}.

The tool uses two critical pieces of information to create a simplified model of the care system.

  1. Placement types
  2. Child Age

Start by exploring SSDA903 data for your region or by creating your base forecast. You can then adjust your forecast by changing entry, exit, or transition rates.

You may save your current scenario at any point during modelling to return to later.


Step 1 - View your historic data

Check historic data from the SSDA903.

You can filter to focus on a particular population.

Time required to complete: 3-5 minutes.

Step 2 - Create your base forecast

Your base forecast is the first future scenario created using trends from historic data

  • Select the population you want to create a forecast for.
  • Select the date you want your forecast to start.
  • Select the time period you want the model to reference
Time required to complete: 4-7 minutes

Step 3 - Adjust your forecast

Adjust the base forecast to create different scenarios.

  • Adjust the flow of children from one placement type to another, or leaving care.
  • Adjust the number of children entering any placement subcohort.
Time required to complete: 5-10 minutes
